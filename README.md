# Assignment 3: Disk Scheduling Algorithms

## Description
This project is a simulation of how an operating system schedules disk access. It tests three different algorithms:
* **FCFS** (First Come First Serve)
* **SSTF** (Shortest Seek Time First)
* **SCAN** (Elevator Algorithm)

The program calculates the "Total Head Movement" for 3 different test cases to see which algorithm is the most efficient and fair.

## How to Run the Program

1. Make sure you have Java installed on your computer.
2. Go to the folder where the file is saved (inside the src folder).
3. Run the program directly by typing:
   `java DiskScheduling.java`

## What to Expect
When you run the program, you do not need to type anything. It will automatically run Case 1, Case 2, and Case 3. It will print the step-by-step service order and the total head movement for all three algorithms.
